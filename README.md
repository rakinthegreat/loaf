# Loaf - Interactive Cat Game

Loaf is a high-performance, visually rich sensory experience designed specifically for feline engagement. Built with Flutter, it features scientifically-tuned game modes that leverage cat-centric color palettes and tactile feedback.

## Game Modes

### 🐟 Fish Tank (New!)
A high-fidelity koi pond simulation featuring:
*   **Procedural AI**: Smooth, arc-based swimming patterns that mimic real fish.
*   **High-Fidelity Visuals**: 3D-shaded Koi with translucent fins and realistic "rays."
*   **Dynamic Water**: Interactive ripples and moving caustic light effects that respond to every tap.

### 🪲 Bug Swarm
A fast-paced hunting game optimized for cat vision:
*   **Cat-Vision Colors**: Ultra-neon yellow, electric blue, and pure white palettes for maximum contrast against green grass.
*   **High-Detail Creatures**: Beetles, moths, cockroaches, and fireflies with 3D bodies and iridescent wings.

### 🐀 Whack-a-Mouse
A classic hunting experience with a premium twist:
*   **Soft-Fur Rendering**: 3D-shaded mice with detailed ears, whiskers, and realistic eyes.
*   **Tactile Feedback**: Strong vibration pulses when the mouse is caught.

### 🔴 Laser Path
A high-speed chase mode:
*   **Glow Dynamics**: Intense glowing points with particle "sparks" on contact.
*   **Physics-Based Motion**: High-velocity movement designed to trigger predatory tracking.

## 🚀 Key Features

*   **Scientifically Tuned Colors**: All games use dichromatic-friendly palettes (focusing on Blues and Yellows) to ensure maximum visibility for cats.
*   **Synthetic Audio Engine**: Custom-generated WAV sounds (splats, splashes, squeaks) for low-latency, high-quality audio feedback.
*   **Custom Vibration System**: Specialized haptic pulses (25ms to 100ms) that provide physical feedback during hunting sessions.
*   **Edge-to-Edge Display**: Full immersion with liquid-glass UI elements and deep blur effects.

## 🛠 Tech Stack

*   **Framework**: Flutter
*   **Rendering**: CustomPainter (Procedural Graphics)
*   **Physics**: Custom Vector Mathematics
*   **Audio**: audioplayers (Synthetic WAV generation)
*   **Haptics**: custom vibration system for Android
